Для создания docker-образа нужно:
Скачайте себе файлы в отдельную папку. Это будет ваша рабочая директория. Первым делом нужно инициализировать в ней Git-репозиторий. Для этого перейдите в директорию и выполните следующие команды:
git init
git commit -m "first commit"
git status
git add .
docker build -t "Ваша рабочая директория" .
docker run -p 8000:8000 "Ваша рабочая директория"

Проверьте работу вашего приложения, выполнив команду в postman: 
Get http://localhost:8000/status
Get http://localhost:8000/version
POST http://localhost:8000/predict
